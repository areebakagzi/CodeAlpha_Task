\# Unemployment Analysis with Python



\## Project Overview



This project analyzes unemployment trends in India using Python. The objective is to explore unemployment patterns, labour participation rates, employment distribution, and differences between urban and rural areas through data analysis and visualization.



\## Dataset Information



The dataset contains information about:



\- Region

\- Date

\- Frequency

\- Estimated Unemployment Rate (%)

\- Estimated Employed

\- Estimated Labour Participation Rate (%)

\- Area (Urban/Rural)



Total Records After Cleaning: 740



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Jupyter Notebook



\## Project Workflow



\### 1. Data Loading

\- Imported dataset using Pandas.

\- Examined dataset structure and columns.



\### 2. Data Cleaning

\- Checked for missing values.

\- Removed incomplete records using `dropna()`.



\### 3. Exploratory Data Analysis

\- Statistical summary of unemployment and employment indicators.

\- Distribution analysis using histograms.



\### 4. Data Visualization

\- Distribution of Unemployment Rate

\- Top 10 Regions with Highest Unemployment Rate

\- Urban vs Rural Unemployment Comparison

\- Unemployment Trend Over Time

\- Labour Participation Rate Distribution

\- Employment Distribution Analysis



\## Key Findings



\- Average unemployment rate was approximately 11.79%.

\- Tripura recorded the highest average unemployment rate (28.35%).

\- Urban unemployment (13.17%) was higher than rural unemployment (10.32%).

\- Average labour participation rate was 42.63%.

\- Unemployment rates showed noticeable fluctuations over time.

\- Employment distribution was highly skewed across regions.



\## Conclusion



The analysis revealed significant variations in unemployment across regions and areas. Urban regions experienced higher unemployment than rural regions, while labour participation remained concentrated around moderate levels. These findings provide useful insights into employment dynamics and workforce participation patterns in India.
