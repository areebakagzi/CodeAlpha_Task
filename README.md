# CodeAlpha Data Science Internship

This repository contains the projects completed during the CodeAlpha Data Science Internship.

## Projects

### 1. Iris Flower Classification

Machine learning project using the K-Nearest Neighbors (KNN) algorithm to classify Iris flowers into different species based on flower measurements.

### 2. Unemployment Analysis

Data analysis project exploring unemployment trends in India using Python, Pandas, and Matplotlib. The project includes data cleaning, visualization, and trend analysis.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Jupyter Notebook
